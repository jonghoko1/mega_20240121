블록과 인라인

요소성질

블록성질 : h1~h6(headline tag - 타이틀 파트), p(paragraph - 문단), div(divide - 공간 형성, 그룹핑 작업)

- 줄바꿈이 존재

- 공간에 대한 구성이 정확하게 가로값과 세로값으로 떨어진다.

- 독립적인 말단 태그(h1 ~ h6, p), 복합적인 태그(div)

블록 성질 => 레고블록(상하로 쌓인다. = 세로방향으로 나열)

인라인성질 : a (anchor - 링크 걸어줄 때 사용 / href), img (image - 이미지를 넣을 때 사용 / src(source), alt(alternative - 시각장애인을 위한 이미지의 설명), span (인라인 성질을 갖고 있는 독립적이면서도 그룹핑이 가능))

- 줄바꿈이 없음

- 자체 커서영역이 존재(통제대상) : HTML 문서상에서 엔터값이 화면상에 반영

- 공간에 대한 세로영역이 모호

- 독립적인 말단 태그(a, img, span), 복합적인 태그(span)

인라인 성질 => 인라인 스케이트의 바퀴(좌우로 쌓인다. = 가로방향으로 나열)

타이틀 h2 - 폰트 크기 24px(1.5em)

타이틀 h3 - 폰트 크기 18.72px(1.17em)

타이틀 h5 - 폰트 크기 13.28px(0.83em)
타이틀 h6 - 폰트 크기 10.72px(0.67em)
