이벤트

이벤트란, 사용자가 일련의 동작을 수행하는 과정에서 함수문을 호출하여 실행시킨 후 결과로서 피드백을 받는 과정

- 요소 대상 이벤트 : 클릭(click) 이벤트, 마우스(mouse) 이벤트, 키보드(keyboard) 이벤트

- 전체 대상 이벤트 : load, resize, scroll

- form 대상 이벤트 : submit, change

이벤트를 진행할 수 있는 함수 호출 방법 - 외부스크립트 연동 시 포함

1. 태그 내부에 on이벤트명 = "함수 호출명();"

2. (in Javascript) 선택자.on이벤트명 = "함수호출문"

3. (in Javascript) 선택자.addEventListner("이벤트명", function(){실행문...})

요소대상 이벤트 : mouseover & mouseout

직접 함수를 호출하여 이벤트를 구성(함수호출을 외부스크립트에 작성하지 않음)

외부스크립트를 활용하여 함수 호출1

외부스크립트를 활용하여 함수 호출2

외부스크립트를 활용하여 함수 호출3

요소대상 이벤트 : click

게임 캐릭터의 진행(소모, 충전)

현재 변경된 상태

요소 대상 이벤트 : keyboard 이벤트

keydown : 키보드를 눌렀을 때 이벤트가 발생한다.

keypress : 키보드를 누른 상태에서 이벤트가 발생한다.

keyup : 키보드를 눌렀다가 떼었을때 이벤트가 발생한다.
